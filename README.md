# Books API 

## Introduction

This is a simple API that allows users to create, read, update, and delete books. It uses a MySQL database to store the data.

## Prerequisites

- Docker
- Docker Compose
- Make

## Installation

1. Clone the repository to your local machine.

```bash
git clone https://github.com/emy123/books-api.git
```

2. Change to the project directory.

```bash
cd books-api
```

3. Build the Docker images.

```bash
make build
```
